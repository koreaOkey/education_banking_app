# Images 폴더

이 폴더는 앱에서 사용되는 이미지 파일들을 저장하는 곳입니다.

## 폴더 구조

```
images/
└── thumbnails/     # 썸네일 이미지
    └── (썸네일 이미지들을 여기에 저장)
```

## 사용 방법

### 썸네일 이미지 사용
```jsx
<img src="/images/thumbnails/your-image.png" alt="설명" />
```

## 이미지 저장 위치

- **썸네일**: `public/images/thumbnails/`
- 필요시 추가 폴더 생성 가능 (예: icons, logos, backgrounds 등)
