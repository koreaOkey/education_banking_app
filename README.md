# Banking App Mockup

신한은행 앱 UI를 참고하여 만든 React 기반 뱅킹 앱 목업입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```

브라우저에서 http://localhost:3000 으로 접속하시면 됩니다.

## 주요 기능

- 📱 모바일 최적화 UI
- 🏦 계좌 정보 표시
- 💳 서비스 그리드 메뉴
- 🔄 반응형 디자인 (모바일/데스크톱)
- 🎨 모던한 UI/UX

## 구조

```
src/
├── App.js                    # 메인 앱 컴포넌트
├── App.css
├── components/
│   ├── Header.js            # 상단 헤더
│   ├── Banner.js            # 프로모션 배너
│   ├── AccountSection.js    # 계좌 정보 섹션
│   ├── QuickMenu.js         # 빠른 메뉴
│   ├── ServiceGrid.js       # 서비스 그리드
│   └── BottomNav.js         # 하단 네비게이션
└── index.js
```

## 기술 스택

- React 18
- CSS3
- Modern JavaScript (ES6+)

## 참고사항

이 프로젝트는 교육 목적의 UI 목업이며, 실제 뱅킹 기능은 구현되어 있지 않습니다.
